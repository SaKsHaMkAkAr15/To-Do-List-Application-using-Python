# To Do List App 

## What it does
A command-line To Do List app built in Python that allows users to add, view, mark as done and delete tasks. All tasks are saved persistently using CSV files.

## Features
1. Add a task
2. View all tasks with status
3. Mark a task as done
4. Delete a task
5. Data persists using CSV storage

## What I got to learn inn this
- Python lists and dictionaries
- File handling with CSV module
- Try/except for error handling
- Enumerate for numbered lists
- Loops and conditionals

## How to run
1. Make sure Python is installed
2. Clone this repository
3. Run this command:
python todo.py
